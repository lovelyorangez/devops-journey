## 📖 Concepts Learned

### 1. Navigation & Directory Structure
Understanding how to move around the Linux Hierarchical Directory Structure.
* **Absolute Path:** Path starting from the root `/` (e.g., `/home/user/Desktop`).
* **Relative Path:** Path relative to your current location (e.g., `../documents`).

### 2. Core Commands
| Command | Description | Key Flag/Usage |
| :--- | :--- | :--- |
| `pwd` | Print Working Directory | Confirms current location. |
| `ls` | List files | `ls -a` to see hidden files (like `.ssh`). |
| `cd` | Change Directory | `cd ~` goes home; `cd -` goes to previous dir. |
| `mkdir` | Make Directory | `mkdir -p` to create nested folders. |
| `mv` | Move / Rename | Used to move files or rename them in place. |
| `cp` | Copy | `cp -r` to copy entire directories. |
